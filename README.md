# 프로젝트 소개

## 프로젝트 개요

대학생들은 시험기간마다 여러 과목을 동시에 준비해야 하지만,
공부 시간은 제한적이며 어떤 과목부터 공부해야 하는지 판단하기 어려운 경우가 많다.

ExamPilot은 이러한 문제를 해결하기 위해 개발한 AI 기반 시험 학습 플래너이다.

단순히 공부 시간을 배분하는 것이 아니라

- 시험까지 남은 기간
- 하루 공부 가능 시간
- 과목별 난이도
- 과목별 중요도
- 공부 스타일

등을 종합적으로 분석하여 개인 맞춤형 학습 계획을 생성한다.

또한 사용자가 중간에 조건을 변경하거나 새로운 정보를 추가하더라도 기존 계획을 유지하면서 필요한 부분만 수정하도록 설계하였다.

## 모델별 특징

### GPT-5.4

장점

- 요구한 출력 형식을 가장 정확하게 준수하였다.
- 과목별 우선순위를 선정한 이유가 가장 구체적이었다.
- 복습 일정과 체크리스트가 현실적으로 작성되었다.
- 출력이 길어 중간에 끊겼지만 "4번부터 이어서 작성" 요청 후에도 문맥을 정확하게 유지하였다.

단점

- 응답 길이가 길어 한 번에 출력되지 않았다.

---

### Claude Opus 4.8

장점

- 문장이 가장 자연스럽고 읽기 쉬웠다.
- 전체 구성이 깔끔하게 정리되어 있었다.
- 출력 형식을 잘 준수하였다.

단점

- GPT보다 학습 전략 설명이 조금 단순하였다.
- 세부 공부법이 다소 간략하였다.

---

### Gemini 3.1 Pro

장점

- 학습 동기부여 문구가 포함되어 친절한 느낌을 주었다.
- Python 공부 팁과 예시 코드 등 부가적인 정보를 제공하였다.
- 전체적인 가독성이 우수하였다.

단점

- 응답 생성 속도가 상대적으로 느렸다.
- 처음 응답 생성 시 로딩 시간이 길었다.
- 문맥 유지 측면에서는 GPT보다 다소 아쉬웠다.

# 최종 선정

최종 선정 모델은 GPT-5.4이다.

선정 이유는 다음과 같다.

첫째, 요구한 출력 형식을 가장 정확하게 준수하였다.

둘째, 과목별 중요도와 난이도를 종합적으로 분석하여 현실적인 학습 계획을 제시하였다.

셋째, 추가 요청에도 기존 내용을 유지하며 자연스럽게 이어서 작성하여 문맥 유지 능력이 우수하였다.

넷째, 정보가 부족한 경우 임의의 사실을 생성하지 않고 제한사항을 먼저 설명하여 환각 가능성을 최소화하였다.

# 시스템 프롬프트

당신은 대학생을 위한 AI 시험 학습 코치이다.

목표는 사용자의 시험 준비를 효율적으로 도와주는 것이다.

다음 원칙을 반드시 따른다.

1. 시험 일정과 공부 가능 시간을 먼저 분석한다.

2. 과목의 중요도와 난이도를 고려하여 우선순위를 결정한다.

3. 현실적으로 실천 가능한 공부 계획만 작성한다.

4. 사용자가 제공하지 않은 정보는 절대로 추측하지 않는다.

5. 정보가 부족하면 최대 3개의 확인 질문을 먼저 한다.

6. 출력은 반드시 아래 형식을 따른다.

- 과목별 우선순위
- 우선순위 선정 이유
- 학습 계획
- 복습 일정
- 하루 공부 순서
- 휴식 방법
- 시험 직전 체크리스트

7. 사실이나 수치를 제시하는 경우에는 근거가 명확해야 하며, 확인이 필요한 내용은 "확인 필요"라고 표시한다.

# 프롬프트 개선

## Version 1

시험 공부 계획을 만들어 주세요.

### 문제점

- 정보가 부족해 결과 품질이 일정하지 않았다.
- 출력 형식이 매번 달라졌다.
- 과목별 우선순위가 불명확했다.

---

## Version 2

1. 입력 정보를 먼저 확인한다.

2. 부족한 정보는 최대 3개까지 질문한다.

3. 과목별 난이도와 중요도를 분석한다.

4. 학습 시간을 계산한다.

5. 복습 계획을 포함한다.

6. 체크리스트를 제공한다.

7. 최종 답변은 핵심만 표 형태로 정리한다.

### 개선 결과

- 출력 형식이 일정해졌다.
- 환각 발생 가능성이 감소하였다.
- 실제 활용성이 높아졌다.
- 사용자가 그대로 따라 하기 쉬운 결과를 제공하였다.

## Pass 기준

- 정답을 알고 있는 경우 근거와 함께 설명한다.
- 정보가 부족하면 "확인이 필요합니다."라고 답한다.
- 부족한 정보를 먼저 질문한다.

## Fail 기준

- 존재하지 않는 시험 범위를 생성한다.
- 사용자가 말하지 않은 정보를 사실처럼 답한다.
- 확인 질문 없이 임의로 계획을 작성한다.

# 심층 인터뷰 문항

## 1. 무료 모델만 사용해야 한다면 품질을 유지하기 위해 어떻게 하겠는가?

무료 모델은 응답 길이와 사용 횟수에 제한이 있으므로 하나의 긴 프롬프트 대신 **정보 확인 → 학습 계획 생성 → 결과 수정**의 3단계로 나누어 요청한다. 또한 입력 템플릿을 재사용하여 응답 형식을 일정하게 유지한다.

## 2. 시험 규정이나 정책처럼 자주 변경되는 정보를 다룰 때 환각을 줄이는 방법은?

최신 강의자료나 공지사항을 함께 입력으로 제공하도록 설계한다. 필요한 정보가 없으면 임의로 답변하지 않고 **"확인이 필요합니다."**라고 안내한 후 사용자가 최신 자료를 제공하도록 유도한다.

## 3. 10턴 이상 대화에서 문맥이 끊기는 문제가 발생한다면 어떻게 개선하겠는가?

시스템 프롬프트에 **이전 대화에서 확정된 조건을 항상 유지한다**는 규칙을 추가한다. 또한 응답을 생성하기 전에 현재까지 확정된

# 최종 결론

ExamPilot은 대학생의 시험 준비를 지원하기 위한 AI 기반 학습 계획 생성 시스템이다.
